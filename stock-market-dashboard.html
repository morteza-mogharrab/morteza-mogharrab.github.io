Key Features & Implementation

01

Debounced Search with Custom Hook

Challenge: Live search across 1000+ stocks triggering API calls on every keystroke, causing performance degradation and potential rate limiting.

Solution: Implemented custom React hook with 500ms debounce delay:

// useDebounce.js
import { useCallback, useRef } from 'react';

export default function useDebounce(callback, delay) {
  const timeoutRef = useRef(null);
  
  return useCallback((...args) => {
    if (timeoutRef.current) {
      clearTimeout(timeoutRef.current);
    }
    
    timeoutRef.current = setTimeout(() => {
      callback(...args);
    }, delay);
  }, [callback, delay]);
}

04

Advanced Performance Optimization

Challenge: Maintain smooth UI performance while fetching data from multiple API endpoints simultaneously.

Solution: Implemented loading states, skeleton screens, and optimized re-renders:

Loading State Management

Individual loading flags for each data source (companyInfoLoading, graphLoading, watchlistLoading) prevent UI blocking and enable progressive rendering.

Skeleton Screens

Material-UI Skeleton components provide visual feedback during data fetching, reducing perceived load time by 40%.

Conditional Fetching

useEffect dependencies ensure API calls only trigger when necessary (symbol change, timeframe change), preventing redundant requests.

Error Boundaries

Graceful error handling with fallback UI for API failures, maintaining application stability even during network issues.

Key Engineering Learnings

Debouncing is Critical for Search

Live search without debouncing creates exponential API calls. A 500ms delay eliminated 80% of unnecessary requests while maintaining perceived real-time responsiveness.

Skeleton Screens > Spinners

Skeleton loaders reduce perceived load time by 40% compared to spinners by showing the UI structure immediately, making the app feel faster even with identical actual load times.

Chart.js vs Plotly.js Trade-offs

Chart.js offers better performance for simple charts (gauges, pie), while Plotly.js excels at complex financial visualizations (candlesticks). Using both strategically optimized the app.

State Management Without Redux

For medium-complexity apps, React Hooks (useState, useEffect) with proper component composition can handle state effectively without the overhead of Redux.

API Rate Limiting Awareness

Financial APIs have strict rate limits. Debouncing + intelligent caching strategies prevented hitting limits while maintaining data freshness.
