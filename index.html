Building Products at Scale

At PetoLab, I've evolved from Frontend Developer to Full-Stack Developer while building production systems that serve real users. As the developer behind PetoKingdom's cross-platform social media app and its internal admin dashboard, I've contributed to architecting comprehensive solutions that handle user management, real-time analytics, content moderation, and feature toggling—all while leveraging AI tools to enhance development efficiency.

My approach combines deep technical expertise with modern development practices. I've built cross-platform applications with React Native, implemented state management with Redux, integrated GraphQL APIs, and architected backend systems with Spring Boot and Docker. What enhances my workflow is how I strategically use AI tools like Cursor AI and CodeRabbit to accelerate development while maintaining code quality and security standards.

Working alongside teams implementing production AI systems—from recommender architectures to content generation features—has given me insights into how intelligent systems integrate with traditional development. This experience shaped my methodology: using AI not as a replacement for engineering excellence, but as a powerful tool that enables developers to focus on architecture, user experience, and solving complex problems.

AI-Enhanced Development Approach

While the industry evolves with AI integration, I've developed a practical approach that enhances traditional development workflows. My toolkit includes intelligent code generation, automated code reviews, and AI-assisted debugging—all while maintaining rigorous testing and security standards.

For example, when implementing enterprise authentication for our admin dashboard, I used AI tools to accelerate research and generate secure boilerplate code, then applied traditional engineering practices for testing and security validation. The result was a production-ready system with JWT rotation and comprehensive audit logging that meets enterprise security standards.

My AI-enhanced workflow includes automated PR reviews for early issue detection, intelligent refactoring for technical debt management, and smart documentation generation for team knowledge sharing. This approach allows me to focus on architecture decisions, user experience design, and complex problem-solving while AI handles repetitive implementation tasks.

I help teams adopt these tools pragmatically—starting with low-risk automation, measuring improvements, then expanding to more complex use cases. The future belongs to developers who effectively combine human creativity with AI capabilities.
